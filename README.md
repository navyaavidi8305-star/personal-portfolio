# Personal Portfolio Website

A full-stack personal portfolio website built using React.js, Node.js, Express.js, and MongoDB Atlas.

## Features

- Responsive portfolio website
- Projects section integrated with MongoDB
- Contact form with database storage
- Full-stack deployment

## Tech Stack

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Frontend: Vercel
- Backend: Render
